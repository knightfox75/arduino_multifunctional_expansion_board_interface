--------------------------------------------------------------------------------

	Arduino Multifunctional Expansion Board Interface (AMEBI)
	Un librería para la gestión fácil de este módulo de expansión.

--------------------------------------------------------------------------------

1. El objetivo
   El principal objetivo de esta librería es facilitar lo máximo posible el uso
   de este módulo de expansión para Arduino, orientado al aprendizaje.
   
2. Que se incluye
   La librería lista para usar en el formato oficial de Arduino, así como el
   codigo fuente de la libreria y ejemplos.
